		loadModel (line 55)
	
 
	
	

Load the model with the given name.


loadModel("SongModel") would include models/songmodel.php and create the object in the controller, like this:  $songs_model = $this->loadModel('SongsModel');  Note that the model class name is written in "CamelCase", the model's filename is the same in lowercase letters
